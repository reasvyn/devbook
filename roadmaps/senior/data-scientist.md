# Senior Data Scientist

## Description

What a senior data scientist should know — advanced modeling, experiment design, ML pipeline architecture, stakeholder communication, and driving data-informed decisions.

## Prerequisites

- Foundational statistics and programming (equivalent of university-level coursework)

## Learning Path

### Advanced Modeling

- `🔴 CRITICAL` Supervised learning — ensemble methods (XGBoost, LightGBM, Random Forest)
- `🔴 CRITICAL` Unsupervised learning — clustering, dimensionality reduction (PCA, t-SNE)
- `🔴 CRITICAL` Time series forecasting — ARIMA, Prophet, LSTMs
- `🔴 CRITICAL` Deep learning — neural networks, CNNs, RNNs, transformers
- `🟠 HIGH` Bayesian methods — probabilistic programming, PyMC
- `🟠 HIGH` Causal inference — A/B testing, instrumental variables, difference-in-differences
- `🟢 LOW` Reinforcement learning — MDP, Q-learning, policy gradients

### Experiment Design & Causal Reasoning

- `🔴 CRITICAL` A/B testing — sample size, power, multiple testing correction
- `🔴 CRITICAL` Experiment design — randomization, stratification, guarding metrics
- `🔴 CRITICAL` Interpreting results — practical vs statistical significance
- `🟠 HIGH` Online controlled experiments — ramp-up, holdout groups
- `🟠 HIGH` Causal inference methods — propensity score matching, DiD

### ML Engineering

- `🔴 CRITICAL` Feature engineering — feature stores, encoding, scaling
- `🔴 CRITICAL` Model evaluation — cross-validation, metrics, bias/variance trade-off
- `🔴 CRITICAL` MLOps — model versioning, deployment, monitoring
- `🟠 HIGH` Data pipelines — Airflow, Prefect, dbt
- `🟠 HIGH` Model serving — REST endpoints, batch inference, edge deployment
- `🟢 LOW` Distributed training — Spark ML, Ray, Horovod

### Data Infrastructure

- `🔴 CRITICAL` SQL for analytics — complex aggregations, window functions, CTEs
- `🔴 CRITICAL` Data warehousing — BigQuery, Snowflake, Redshift
- `🟠 HIGH` Data lake vs warehouse vs lakehouse
- `🟠 HIGH` Streaming data — Kafka, Kinesis, real-time features

### Communication

- `🔴 CRITICAL Translating business questions into analytical problems
- `🔴 CRITICAL Communicating uncertainty — confidence intervals, error margins
- `🔴 CRITICAL Data visualization — telling stories with charts (Matplotlib, ggplot, Tableau)
- `🟠 HIGH Working with product and engineering — scoping, prioritization
- `🟠 HIGH Stakeholder management — getting buy-in for data initiatives

### Ethics & Governance

- `🔴 CRITICAL` Bias in ML — detecting and mitigating algorithmic bias
- `🔴 CRITICAL` Data privacy — GDPR, CCPA, data anonymization
- `🟠 HIGH` Model fairness — fairness metrics, disparate impact
- `🟠 HIGH` Explainability — SHAP, LIME, interpretable ML

### Leadership

- `🔴 CRITICAL Defining data science roadmap — prioritizing high-impact problems
- `🔴 CRITICAL Mentoring junior and mid data scientists
- `🟠 HIGH Building data culture — data literacy across the organization
- `🟠 HIGH Cross-functional collaboration — data engineering, ML engineering, product
