# What Is Discrete Mathematics

## Description

Discrete mathematics studies objects that are countable or distinct — integers, graphs, logical statements. Unlike calculus (which deals with continuous change), discrete math is the natural language of computer science: bits, data structures, algorithms, and state machines are all discrete.

## Content / Material

### Continuous vs. Discrete

| Continuous (Calculus) | Discrete |
|-----------------------|----------|
| Real numbers | Integers |
| Smooth change | Step changes |
| Area under a curve | Sum of a sequence |
| Limits and infinity | Finite or countably infinite |

Computers are inherently discrete — they operate on bits and integers. Discrete math is the natural fit.

### The Branches That Matter Most

| Branch | What It Studies | Developer Relevance |
|--------|----------------|---------------------|
| **Logic** | Propositions, truth, inference | All conditionals, type systems, contracts |
| **Set theory** | Collections, membership, operations | SQL, JSON, database queries |
| **Combinatorics** | Counting, permutations, combinations | Algorithm analysis, password strength |
| **Graph theory** | Nodes, edges, paths | Social networks, routing, dependency resolution |
| **Number theory** | Integers, primes, modular arithmetic | Cryptography, hashing, checksums |

### Where It Shows Up in Code

- **Algorithms**: graph traversal, shortest paths, sorting networks.
- **Data structures**: trees, hash tables, graphs — all discrete structures.
- **Databases**: SQL queries are set operations on relations.
- **Cryptography**: RSA is built on number theory (primes, modular exponentiation).
- **Networking**: routing algorithms are graph theory problems.
- **Compilers**: parsing is applied formal language theory (automata, grammars).

### The Mindset: Counting and Structure

Discrete math teaches you to reason about:

- **Counting**: how many possible passwords of length 8? How many paths through this graph?
- **Structure**: what's the relationship between these elements? Is it a tree, a graph, a partial order?
- **Existence**: does a solution exist? Is it unique?
- **Optimization**: which arrangement gives the best outcome?

## Glossary

| Term | Definition |
|------|------------|
| Discrete | Separate, distinct, countable — not continuous |
| Graph | A collection of nodes and edges connecting them |
| Combinatorics | The branch of math dealing with counting and arrangements |
| Modular arithmetic | Arithmetic where numbers wrap around after reaching a modulus |
| Set | A collection of distinct elements |
