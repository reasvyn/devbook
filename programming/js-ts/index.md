## Glossary

| Term | Definition |
|------|------------|
| API | Application Programming Interface — a set of methods and properties exposed by the browser or runtime |
| Async | Short for asynchronous — code that runs out of the main execution order |
| DOM | Document Object Model — the browser's tree representation of HTML |
| Event Loop | The mechanism that handles async callbacks in JavaScript |
| Promise | An object representing the eventual completion or failure of an async operation |
| TypeScript | A typed superset of JavaScript that compiles to plain JavaScript |
